HushSecurity WordPress Theme

A modern landing page theme for WordPress.

== Description ==

HushSecurity is a professional WordPress theme designed for security companies and SaaS platforms. It features a modern design with purple and red accents, fully responsive layout, and optimized for performance.

== Installation ==

1. Upload the theme folder to /wp-content/themes/
2. Activate the theme through the 'Themes' menu in WordPress
3. Configure menus in Appearance → Menus
4. Customize in Appearance → Customize

== Features ==

* Responsive design
* Custom navigation menus
* Custom logo support
* Google Fonts integration (Figtree & Inter)
* SVG graphics support
* Modern CSS Grid layout
* Optimized for speed
